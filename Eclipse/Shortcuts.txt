Refactoring
------------
Alt-Shift-R = rename class, variables, methods
Alt-Shift-T = Generic refactoring options (Move, Interface, etc)
Alt-Shift-M = extract a method (creates a new method in place of code)
Alt-Shift-I = Inline method OR replace method invocation with actual method
Alt-Shift-C = add parameter to method (affects all classes that call method)

----------

Alt-Shift-I = Inline the variable (replace...)
